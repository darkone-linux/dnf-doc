### User Profiles

Each user is linked to a "profile" (choice and interchangeable), which determines:

* The programs and configurations for [machines](/en/doc/user-guide/linux-workstations/) (education, office, games...).
* Access and configuration for [online services](/en/doc/user-guide/services/).

| **Profile** | **Description** |
|----|----|
| ⚪ minimal | The strict minimum of programs and tools. |
| 🔵 normal | For non-IT office user -> the essentials. |
| 🟣 advanced | For IT professional -> a nice shell and development tools. |
| 🔴 admin | Account dedicated to network and systems administrators (expert). |
| 🎨 creator | For multimedia creator -> video, music, photo, image. |
| 📘 student | For studies -> taking notes, organization, studying. |
| 🎮 Gamer | Plenty of native Linux games. |
| 🎒 teen | Educational and fun apps to start with internet. |
| 🧩 child | Educational software to learn, without internet access. |
| 🧸 baby | Very stripped system -> learning mouse, numbers, shapes, etc. |
