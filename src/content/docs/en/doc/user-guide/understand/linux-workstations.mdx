The cloud Linux system installs on any PC connected to the network.

:::note[Linux, is it complicated?]
Contrary to popular belief, **Linux and its applications have become very easy to use.**
And our pre-installed system is even easier to get started with!

**Why Linux / NixOS?**

- **Linux** is free, secure, and aligned with the idea of a sovereign and secure digital space.
- **NixOS** is an innovative Linux: declarative, reproducible, and very stable.
- Open and maintained code ensures the lowest risk of "backdoors" in the world.
:::

:::tip[Time savings at every level]
- 100% pre-installed → programs, user accounts and configurations, everything is there.
- 100% IT-managed → no need to administer the system, everything is automatic.
- Ideal and performant with old PCs you no longer use.
:::

## Workstation profiles

Each machine is configured according to a **profile** (role) that includes:

- **Features** enabled automatically.
- **Pre-made configurations**.

| | Profile | Role |
|---|---|---|
| ⚙️ | `minimal` | A system with the bare minimum. |
| 🖥️ | `desktop` | A desktop with office, multimedia, and productivity software. |
| 💻 | `laptop` | An office workstation optimized for laptops. |
| 🔌 | `portable` | My system on a bootable USB drive. |
| 🗄️ | `server` | A stripped-down, secure server without a screen. |
| 🌐 | `gateway` | A zone gateway, my internet "box" that filters ads. |
| ☁️ | `hcs` | A VPN coordination server, for the administrator. |
| 📦 | `vm` | A virtual machine. |

Technical details&nbsp;: [Host profiles](/en/doc/admin-guide/understand/host-profiles/).

:::tip[My profile is customizable!]
- I can choose it at installation and later **change it at any time**.
- I can add **personal programs and configurations** to it.
:::

## Zero maintenance

Under DNF, the system manages itself&nbsp;:

- **Updates**: applied automatically, without intervention.
- **Security**: Linux is secure by design → no antivirus needed.
- **Drivers**: everything is pre-configured. Printers, screens, peripherals&nbsp;: they work.
- **Failure or problem**: the admin restores your workstation in minutes, exactly as before.

:::tip[I want to reinstall]
New computer, theft or major technical issue? We can reinstall everything&nbsp;:

- My software
- My configurations
- My data

⚠️ Provided my configuration is declared and my data is backed up.
:::
