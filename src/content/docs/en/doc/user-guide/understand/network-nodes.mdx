### Nodes

:::note[What is a node?]
A network is composed of computers called "nodes". Each has its usefulness.
This information is not necessary to use the cloud but helps understand how it works.
:::

{/* t:p 1e1b419f0bc703497a3f6475f49f876489125a620632b2c8e7f8434c5c5a5479 */}
#### NixOS nodes

They are 100% IT-managed (installed and updated automatically):

| **Type** | **Purpose** |
|----|----|
| ❄️ [Top-notch Linux machines](/en/doc/user-guide/understand/linux-workstations/) | Complete workstation, configured, secured, shared.<br/>→ I just need to log in and work, play, enjoy. |
| 🗄️ Server | A computer that stays on to host services:<br/>→ Jellyfin, Immich, Nextcloud, backup, shares, etc. |
| 🌐 Gateway | A little gem that bridges the local network and the Internet.<br/>→ Firewall, router, DNS server, VPN client… |
| 🎼&nbsp;Coordination&nbsp;server | This shadowy figure is the network's « conductor ».<br/>→ Somewhere on the internet, always ready to serve us. |

:::note[Learn more]
About the network composition in [its presentation](/en/doc/#the-dnf-network).
:::

#### Attached Nodes

I can connect third-party devices to the network:

- 📱 **My smartphone, tablet**<br/>I can connect to the network and all its services.
- 💻 **My computer / beloved system**<br/>Under Windows, macOS or Linux, I also access services.
