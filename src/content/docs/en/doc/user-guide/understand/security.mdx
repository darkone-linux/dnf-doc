My photos, my messages, my documents — my life, in short. Here, they are
protected **from the start**, without the slightest effort on my part. This page
explains, simply, why my system and my services are safer
than elsewhere. And why I can rest easy.

{/* t:p 66ec4b58b73b5d3538e7c80a8d56183b6df253ab9ec5f06ddcbe78fcce096f4b */}
## A Well-Guarded Home

My system is like a home. Security isn't a single big lock&nbsp;: it's a series
of protections that complement each other. If one fails, the others hold.

<CardGrid>
  <Card title="Reinforced Door" icon="seti:lock">
    A firewall filters what comes in. By default, **everything is closed**,
    except what really needs to be open.
  </Card>
  <Card title="Strong Keys" icon="seti:license">
    My passwords are stored in a safe, unreadable to a thief, even if they get
    their hands on it.
  </Card>
  <Card title="Separated Rooms" icon="puzzle">
    Each service lives in its own room. A problem in one doesn't spread to the
    others.
  </Card>
  <Card title="Encrypted Drives" icon="database">
    If the hardware is stolen, the content remains **unreadable** without
    my key.
  </Card>
</CardGrid>

{/* t:p 7ee1e8074a4d95b994ea04d281a42bde0dcbfffff3bf728753478ce7f6baeee0 */}
## Why It's Safer Than Elsewhere

<CardGrid>
  <Card title="Official Rules" icon="document">
    Hardening follows the recommendations of **ANSSI**, the French national
    cybersecurity agency. The same requirements as for sensitive
    administrations.
  </Card>
  <Card title="Audited Software" icon="seti:html">
    Everything is built on **free software** : its code is public, verifiable
    by anyone. Nothing is hidden, no backdoors.
  </Card>
  <Card title="At Home!" icon="nix">
    My data stays on **my machines**, not on the servers of a web giant that
    analyzes it.
  </Card>
  <Card title="Always Up to Date" icon="rocket">
    The system is kept updated **consistently** : security patches arrive
    without breaking my workflows.
  </Card>
</CardGrid>

## What this means for me

Good news&nbsp;: almost nothing changes day-to-day. Security works in the
**background**.

| I want to… | What happens |
|---|---|
| Log in | As usual, just a bit more secure |
| Use my services | Nothing changes, everything stays smooth |
| Choose a password | I'm asked to make it reasonably strong |
| Step away for a bit | The screen locks itself |

## My Role, in Three Actions

The best lock is useless if I leave the key in the door. A few reflexes are
enough.

<CardGrid>
  <Card title="Strong Password" icon="approve-check-circle">
    Long and unique. An easy-to-remember phrase is better than a complicated
    word.
  </Card>
  <Card title="Locked Screen" icon="desktop">
    When leaving my desk, I lock. The system does it for me too.
  </Card>
  <Card title="Ask" icon="comment">
    An unexpected message or request&nbsp;? Better to ask my administrator than
    to click.
  </Card>
</CardGrid>

## In summary

I don't have to become an expert to be well protected. The system handles it,
with **layered** protections, **official rules** and
**transparent software**. My data belongs to me, and it's in good
hands&nbsp;: my own.
