### User profiles

Each user is linked to a "profile" (chosen and interchangeable), which determines:

* The programs and configurations [of workstations](/en/doc/user-guide/understand/linux-workstations/) (education, office, gaming…).
* Access to and configuration of [online services](/en/doc/user-guide/understand/services/).

| **Profile** | **Description** |
|----|----|
| ⚪ minimal | The bare minimum of programs and tools. |
| 🔵 normal | For non-technical office users -> the essentials. |
| 🟣 avancé | For developers -> a nice shell and development tools. |
| 🔴 admin | Account dedicated to network and system administrators (expert). |
| 🎨 créateur | For multimedia creators -> video, music, photo, image. |
| 📘 étudiant | For studies -> taking notes, staying organized, studying. |
| 🎮 Gamer | Plenty of native Linux games. |
| 🎒 Teen | Educational and fun apps to start with the internet. |
| 🧩 Child | Educational software for learning, without internet access. |
| 🧸 Baby | Very stripped-down system -> learning mouse, numbers, shapes, etc. |
