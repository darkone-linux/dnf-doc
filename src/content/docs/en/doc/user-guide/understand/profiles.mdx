### User profiles

Each user is linked to a "profile" (choice-based and interchangeable), which determines&nbsp;:

* The programs and configurations of [workstations](/en/doc/user-guide/understand/linux-workstations/) (education, office, gaming…).
* Access and configuration of [online services](/en/doc/user-guide/understand/services/).

| **Profile** | **Description** |
|----|----|
| ⚪ minimal | The bare minimum of programs and tools. |
| 🔵 normal | For non-IT office users → the essentials. |
| 🟣 avancé | For IT professionals → a nice shell and development tools. |
| 🔴 admin | Account dedicated to network and system administrators (expert). |
| 🎨 créateur | For multimedia creators → video, music, photo, image. |
| 📘 étudiant | For studies → taking notes, organizing, studying. |
| 🎮 joueur | Lots of native Linux games. |
| 🎒 ado | Educational and fun apps to get started with the internet. |
| 🧩 enfant | Educational software for learning, without internet access. |
| 🧸 baby | Very streamlined system → learning the mouse, numbers, shapes, etc. |
