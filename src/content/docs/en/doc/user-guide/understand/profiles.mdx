### User profiles

Each user is linked to a "profile" (chooseable and interchangeable), which determines:

* The programs and configurations of [workstations](/en/doc/user-guide/understand/linux-workstations/) (education, office, gaming…).
* Access and configuration of [online services](/en/doc/user-guide/understand/services/).

| **Profile** | **Description** |
|----|----|
| ⚪ minimal | The bare minimum of programs and tools. |
| 🔵 normal | For non-IT office user → the essentials. |
| 🟣 advanced | For IT professionals → a nice shell and development tools. |
| 🔴 admin | Account dedicated to network and system administrators (expert). |
| 🎨 creator | For multimedia creators → video, music, photo, image. |
| 📘 student | For studies → taking notes, staying organized, studying. |
| 🎮 gamer | Plenty of native Linux games. |
| 🎒 teen | Fun and educational apps to start with the internet. |
| 🧩 child | Educational software for learning, without internet access. |
| 🧸 baby | Very stripped-down system → learning mouse, numbers, shapes, etc. |
