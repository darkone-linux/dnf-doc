### Nodes

:::note[What is a node?]
A network is composed of computers called "nodes". Each has its usefulness.
This information is not necessary to use the cloud but helps understand how it works.
:::

{/* t:p c57a30c94b59b781d8919dd7895d058cd1fad2c7b55ff712bb3cdf74d4f622dc */}
#### NixOS Nodes

They are 100% self-managed (automatically installed and updated):

| **Type** | **Usefulness** |
|----|----|
| ❄️ [Top Linux PC](/en/doc/user-guide/linux-workstations/) | Complete, configured, secure, shared workstation.<br/>→ I just need to connect and work, play, entertain myself. |
| 🗄️ Server | A computer that stays on to host services:<br/>→ Jellyfin, Immich, Nextcloud, backup, shares, etc. |
| 🌐 Gateway | a small gem that connects local network to Internet.<br/>→ Firewall, router, DNS server, VPN client… |
| 🎼&nbsp;Coordination&nbsp;Server | This dark individual is the "conductor" of the network.<br/>→ Somewhere on internet, always ready to serve us. |

:::note[Learn more]
On the network composition in [its presentation](/fr/doc/introduction/#le-réseau-dnf).
:::

#### Attached Nodes

I can connect third-party devices to the network:

- 📱 **My smartphone, tablet**<br/>I can connect to the network and all its services.
- 💻 **My computer / beloved system**<br/>Under Windows, macOS or Linux, I also access services.
