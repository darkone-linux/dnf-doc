### Nodes

:::note[What is a node?]
A network is composed of computers called "nodes". Each has its usefulness.
This information is not necessary to use the cloud but helps understand how it works.
:::

{/* t:p 8aecd0e05907a6ae2792cb517dd4fbd2d9493f173a196f1a64f6ef79cbce5325 */}
#### NixOS nodes

They are 100% IT-managed (installed and updated automatically):

| **Type** | **Purpose** |
|----|----|
| ❄️ [Top Linux workstation](/en/doc/user-guide/presentation/linux-workstations/) | Complete, configured, secure, shared workstation.<br/>→ I just log in and work, play, enjoy entertainment. |
| 🗄️ Server | A computer that stays on to host services:<br/>→ Jellyfin, Immich, Nextcloud, backup, file shares, etc. |
| 🌐 Gateway | A little gem that bridges local network and the Internet.<br/>→ Firewall, router, DNS server, VPN client… |
| 🎼&nbsp;Coordination&nbsp;server | This shady individual is the network's "conductor".<br/>→ Somewhere on the internet, always ready to serve us. |

:::note[Learn more]
About the network composition in [its presentation](/en/doc/introduction/#the-dnf-network).
:::

#### Attached Nodes

I can connect third-party devices to the network:

- 📱 **My smartphone, tablet**<br/>I can connect to the network and all its services.
- 💻 **My computer / beloved system**<br/>Under Windows, macOS or Linux, I also access services.
