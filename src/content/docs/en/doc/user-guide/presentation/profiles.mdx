### User Profiles

Each user is linked to a "profile" (selectable and interchangeable), which determines:

* The programs and configurations of [workstations](/en/doc/user-guide/presentation/linux-workstations/) (education, office, gaming…).
* Access and configuration of [online services](/en/doc/user-guide/presentation/services/).

| **Profile** | **Description** |
|----|----|
| ⚪ minimal | The bare minimum of programs and tools. |
| 🔵 normal | For non-technical office user -> essentials. |
| 🟣 advanced | For IT professional -> a nice shell and development tools. |
| 🔴 admin | Account dedicated to network and system administrators (expert). |
| 🎨 creator | For multimedia creator -> video, music, photo, image. |
| 📘 student | For studies -> taking notes, organizing, studying. |
| 🎮 gamer | Plenty of native Linux games. |
| 🎒 teen | Educational and fun apps to get started with the internet. |
| 🧩 child | Educational software to learn, without internet access. |
| 🧸 baby | Very stripped-down system -> learn mouse, numbers, shapes, etc. |
