## Signing in

I need:

* **A master password** for my vault (vaultwarden).
* **A unique identity** for all cloud services (kanidm / IDM).

My administrator must perform a procedure so I can set these up.

* **Master password**: an email with a link to enter the password.
* **Identity**: a QR code or a link to set a key and a "2FA" password.

:::tip[Good to know!]
- 🔒 **My master password** (Vaultwarden / Bitwarden)<br/>lets me log into my vault.
- 🔑 **An authentication key** (IDM Key or [YubiKey](#using-my-yubikey))<br/>lets me log into all cloud applications.
- 🔐 **A second authentication factor** ([2FA](#adding-a-2nd-factor))<br/>strengthens a password-based access with an extra step.
:::

The **master password** is essential, it's the only one I **absolutely must remember**.

## Using my YubiKey

My YubiKey is a physical key that replaces the password in most situations:

- **Encrypted computer**: at startup, I touch the YubiKey when the LED
  flashes → the drive unlocks by itself, without a passphrase.
- **Logging into my session**: I touch the YubiKey on the login
  screen instead of typing my password.
- **Kanidm (SSO)**: my administrator sends me a registration link.
  I visit it once, I create my passkey with the YubiKey → then I touch the
  key to log in to all cloud services.

:::note[What is a key?]
A digital key lets me identify myself without using a password.
It is stored in my vault.
When an application needs it, Bitwarden opens automatically.
It is the **SIMPLEST AND MOST SECURE** way to authenticate myself.
:::

:::tip[YubiKey = the simplest]
Once registered, I only have to touch my key. No password to
remember or code to copy.
:::

## Adding a 2nd factor

We often add a 2nd factor to a password to secure it.
Nowadays, a huge number of passwords are compromised, which is why this type
of authentication is becoming necessary.

:::tip[2-factor authentication (2FA) combines]
- A password (what I know)
- AND another "factor" (what I have)
:::

The 2nd factor can be a code sent by email / SMS, a 6-digit code generator, etc.

:::caution[Securing my Vaultwarden vault]
A "simple" master password is allowed by default.
But it is highly recommended to add a 2nd factor to secure access,
especially since I put all my passwords in it (bank, social accounts, etc.).
:::
