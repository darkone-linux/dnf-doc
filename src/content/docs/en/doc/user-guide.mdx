## My Sovereign and Secure Cloud

In summary:

- A **practical and secure digital space** for my data.
- **Useful services**: messaging, visio, documents, media, backups, games...
- **Pre-installed Linux systems** (optional) very easy to use.

:::tip[Ideal for children and teens]
A protected space suitable for youngers, for a controlled learning of computer tools and networks!
:::

---

### The Benefits

* 🔐 **Maximum security**: our data remains controlled and transit encrypted.
* 🚫 **No data leaks** to advertisers, AI, insurance, taxes...
* 🔁 **Automated 3-2-1 backups**, encrypted, distributed, compressed.
* 🕐 **Top applications**, stable, useful and simple, to save time.
* 🔑 **Single password** to remember (SSO + vault).
* 👶 **User-friendly** for small and big.

:::note[My digital heritage]
Photos, videos, stories, contacts, dialogues, passwords... more and more memories and data related to my heritage are now digital.
Using a safe, sovereign and controlled place for this data, which are part of my identity, is primordial. This is the purpose of this cloud.
:::

:::tip[Do I need to leave the GAFAM?]
No. These companies offer useful services. Thousands of engineers work there, making the simplicity and efficiency of their tools hard to rival.
They also give a lot to the open-source world and are sometimes valuable partners.
However their private nature does not allow them to guarantee a total and safe space for all my personal data and conversations.
This data can be analyzed and potentially transmitted to advertisers, insurance, AI engines, government services.
Technically, my data can also be confiscated, diverted and kept against my will, even if a serious professional would not do that.
So it's up to me to make the distinction.
:::

### Linux Machines (NixOS)

The cloud's Linux system installs on any PC computer connected to the network.

:::caution[Is Linux complicated?]
Contrary to what people think, **Linux and its applications have become very simple to use.**
And our pre-installed system is even easier to use!
:::

:::tip[A time saver at every level]
* 100% pre-installed -> programs, user accounts and configurations, everything is there.
* 100% self-managed -> no need to administer the system, everything is automatic.
* Ideal and performant with old PCs we no longer use.
:::

:::note[Why Linux / NixOS?]
- **Linux** is a free system, therefore safe, in line with the idea of a sovereign and secure digital space. Linux code transparency guarantees the absence of "backdoors" and potentially unwanted programs.
- **NixOS** is an innovative Linux system that allows reproducibility, meticulous configurations and great stability.
:::

## Connecting and Managing My Account

I need:

* **A main password** for my vault (vaultwarden).
* **A unique identity** for all cloud services (kanidm / IDM).

My administrator must perform a procedure so I can define these accesses.

* **Main password**: an email containing a link to enter the password.
* **Identity**: a QR-Code or a link to define a key and a "2FA" password.

:::tip[To know!]
- 🔒 **My main password** (Vaultwarden / Bitwarden)<br/>allows me to connect to my vault.
- 🔑 **My authentication key** (IDM Key)<br/>allows me to connect to all cloud applications.
- 🔐 **My "2 factors" password** (2FA Key)<br/>allows me to connect too, if I have a problem with my key.
:::

The **main password** is crucial, it's the only thing that needs to be **absolutely remembered**.

My identification follows this schema:

![Identification workflow](../../../../assets/identification.png)

:::note[What is a key?]
A digital key allows identification without using a password.
It is located in my vault.
When an application needs it, Bitwarden opens automatically.
This is the **SIMPLEST AND MOST SECURE** way to authenticate.
:::

:::note[A "2 factors" password?]
2-factor authentication (2FA) uses:
- a password (something I know)
- AND another "factor" (something I possess)

The latter can be a code sent by email/sms, a 6-digit code generator, etc.
These days, a lot of passwords are compromised, which is why this type of authentication is becoming necessary.
:::

:::caution[Securing my vault]
To facilitate access to the vault (vaultwarden / bitwarden), a "simple" main password is allowed by default.
But it is highly recommended to add a 2nd factor to secure access, especially since I put all my passwords in it (bank, social accounts, etc.).
:::

## Online Services

All cloud services are open-source and have their own online documentation. But here are some useful introductions.

### Vaultwarden

Vaultwarden is my vault. It contains my passwords, digital keys, credit cards and confidential information.
This information is stored encrypted and backed up automatically. By default, I am the only one who has access.

:::tip[Useful features]
- Share access with others thanks to "collections".
- Store my passwords and keys in folders.
- Send confidential information thanks to "sends".
:::

:::note[Bitwarden: the client application]
Bitwarden can be installed as a **plugin in a browser** or as a **mobile application**.
It simplifies my life thanks to automatic entry of my credentials and passwords and many other features.
Bitwarden can do everything Vaultwarden can (the server, which holds the data).
:::

:::caution[Important: Bitwarden configuration]
By default, Bitwarden is set to "bitwarden.com".
You must select "self-hosted" then enter the Vaultwarden URL to be able to use Bitwarden with the cloud.

⚠️ I MUST NOT ENTER MY MAIN PASSWORD BEFORE DOING THIS.
:::

### Kanidm (idm)

My identity manager, which allows me to easily identify myself to cloud services.

### Immich (photos)

My personal photo and video manager, with sovereign and effective AI search and facial recognition functions.

To sync photos from my smartphone to Immich:

* Install the immich application on the smartphone.
* Configure sync by giving the Immich URL.

### Nextcloud (files & data)

A space for my files, contacts, calendars and many other personal data, that I can sync and share.

### Matrix (messages & visio)

My social network. All my conversations are encrypted and stay at home. I can link it to the federation (other matrix networks) or not.
I can also sync my WhatsApp, Messenger, Telegram, Signal, Discord accounts...

* Install "Element" on PC and Element Classic on android / ios:
  * Access the network matrix.domaine.tld (don't go to matrix.org!)
  * Connect with "IDM" (don't create an account!)
  * The matrix account is created on first access
  * Invite the user to channels
* For us to be able to call (like with whatsapp), notifications must be enabled and authorized.

:::caution[Select the right matrix server!]
Click on "Self-Hosted" then enter the cloud's matrix domain.
:::

### Jellyfin (films & musics)

My personal media library for my films, my music, that I can use with my television.

### Open WebUI (AI)

A local and sovereign generative AI, based on Open WebUI and Ollama, with local models and secure local storage of histories.

### Outline (notes)

A convenient space for personal or shared notes, writing documentations, doing check-lists.

### Mealie (cooking recipes)

A very convenient app to import, enter, refine my cooking recipes.

### Searxng (search)

A multi-search engine that crosses information from many other engines and allows secure searches.

### AdguardHome (firewall anti-ads)

AdguardHome is hosted on gateways and removes ads and trackers, while isolating the zone with a firewall. Thanks to it, I browse faster and better on the internet.

### Forgejo (git forge)

Forgejo is a git forge, a personal and sovereign GitHub. It allows hosting projects, managing issues, pull requests and much more.
GIT is also an excellent way to organize, version and backup your documents.
