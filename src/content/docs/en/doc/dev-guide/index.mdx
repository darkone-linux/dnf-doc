understand the architecture, create or modify features, and evolve the
documentation.

- [Modules](/en/doc/dev-guide/modules/) — NixOS module organization and creating a new module.
- [Creating a "service" module](/en/doc/dev-guide/create-service-module/) — complete guide: reverse proxy, persistence, firewall, SSO Kanidm, activation.
- [The generator](/en/doc/dev-guide/generator/) — from `config.yaml` to generated Nix code.
- [Contributing to documentation](/en/doc/dev-guide/documentation/) — this Astro/Starlight site, its rules, and its tools.
