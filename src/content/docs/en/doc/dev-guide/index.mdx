This section is intended for developers and AI agents contributing to the
framework: understand the architecture, create or modify features,
and evolve the documentation.

- [Architecture du projet](/en/doc/dev-guide/architecture/) — where things are, from abstraction layers to files.
- [Les modules](/en/doc/dev-guide/modules/) — NixOS module organization and creating a new module.
- [Créer un module de type "service"](/en/doc/dev-guide/create-service-module/) — reverse proxy, persistence, firewall,
  Kanidm SSO, activation.
- [Le générateur](/en/doc/dev-guide/generator/) — from `config.yaml` to generated Nix code.
- [Contribuer à la documentation](/en/doc/dev-guide/documentation/) — this Astro/Starlight site, its rules and tools.
