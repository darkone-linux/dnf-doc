### &#x1F451; darkone.admin.nix

NixOS configuration for the local LAN administrator computer.

* **enable** `bool` Enable NIX configuration builder tools
* **enableNh** `bool` Enable nix helper (nh) management tool

```nix
darkone.admin.nix = {
  enable = false;
  enableNh = false;
};
```
