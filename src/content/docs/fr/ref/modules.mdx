## Modules mixin

:::note
**Un module mixin** définit une collection de modules standard avec une configuration commune cohérente.
:::

### &#x1F4E6; darkone.host.desktop

Une configuration de bureau complète avec gnome, outils multimedia et bureautique.

* **enable** `bool` Configuration d'hôte optimisée pour le bureau

```nix
darkone.host.desktop.enable = false;
```

<hr/>

### &#x1F4E6; darkone.host.gateway

La passerelle / routeur principal d'une zone de réseau local.

:::tip[Une passerelle prête à l'emploi!]
La passerelle est configurée dans le fichier `usr/config.yaml`.
Les services supplémentaires activés (homepage, adguardhome, forgejo, ncps...)
sont automatiquement configurés avec une infrastructure réseau cohérente sur la
passerelle et toutes les machines du réseau local.
:::

* **enable** `bool` Activer les fonctionnalités de passerelle pour l'hôte actuel (dhcp, dns, proxy, etc.)
* **enableFail2ban** `bool` Activer le service fail2ban
* **enableAdguardhome** `bool` Activer le service AdGuard Home pré-configuré
* **enableIdm** `bool` Activer le gestionnaire d'identité (kanidm)

```nix
darkone.host.gateway = {
  enable = false;
  enableFail2ban = true;
  enableAdguardhome = builtins.hasAttr "adguardhome" host.services;
  enableIdm = builtins.hasAttr "idm" host.services;
};
```

### &#x1F4E6; darkone.host.minimal

Toute configuration d'hôte est basée sur cette configuration minimale.

:::caution[Déclaration des services]
Un certain nombre de services (immich, nextcloud, forgejo, etc.) peuvent être déclarés dans la configuration `usr/config.yaml`
de chaque hôte, quel que soit son type (serveur, ordinateur portable, bureau, etc.). **Il est recommandé de les déclarer dans le
fichier yaml pour que le service soit visible sur tout le réseau!**
:::

* **enable** `bool` Configuration minimale d'hôte
* **secure** `bool` Préférer les options plus sécurisées (désactiver les utilisateurs mutables...)
* **enableHomepage** `bool` Activer le service homepage auto-configuré
* **enableForgejo** `bool` Activer le service forgejo git pré-configuré
* **enableImmich** `bool` Activer le service immich pré-configuré
* **enableNextcloud** `bool` Activer le service nextcloud pré-configuré
* **enableMonitoring** `bool` Activer le service de monitoring pré-configuré (prometheus, grafana)
* **enableVaultwarden** `bool` Activer le service Vaultwarden pré-configuré
* **enableNfsHomeShares** `bool` Activer les partages homes NFS DNF
* **enableJitsiMeet** `bool` Activer le service jitsi-meet
* **enableRestic** `bool` Activer le service Restic
* **enableJellyfin** `bool` Activer le serveur Jellyfin
* **enableOutline** `bool` Activer le service Wiki Outline
* **enableDocs** `bool` Activer le service LaSuite Docs
* **enableMealie** `bool` Activer le service Mealie
* **enableSearx** `bool` Activer le service Searx
* **enableMatrix** `bool` Activer un serveur matrix
* **enableTurn** `bool` Activer un serveur coturn (STUN/TURN) pour VOIP
* **enableElement** `bool` Activer l'interface Element-Web (matrix)
* **enableAi** `bool` Activer l'environnement IA (ollama, open-webui, llms...)

```nix
darkone.host.minimal = {
  enable = false;
  secure = false;
  enableHomepage = attrsets.hasAttrByPath [ "services" "homepage" ] host;
  enableForgejo = attrsets.hasAttrByPath [ "services" "forgejo" ] host;
  enableImmich = attrsets.hasAttrByPath [ "services" "immich" ] host;
  enableNextcloud = attrsets.hasAttrByPath [ "services" "nextcloud" ] host;
  enableMonitoring = attrsets.hasAttrByPath [ "services" "monitoring" ] host;
  enableVaultwarden = attrsets.hasAttrByPath [ "services" "vaultwarden" ] host;
  enableNfsHomeShares = attrsets.hasAttrByPath [ "services" "nfs" ] host;
  enableJitsiMeet = builtins.hasAttr "jitsi-meet" host.services;
  enableRestic = builtins.hasAttr "restic" host.services;
  enableJellyfin = builtins.hasAttr "jellyfin" host.services;
  enableOutline = builtins.hasAttr "outline" host.services;
  enableDocs = builtins.hasAttr "docs" host.services;
  enableMealie = builtins.hasAttr "mealie" host.services;
  enableSearx = builtins.hasAttr "searx" host.services;
  enableMatrix = attrsets.hasAttrByPath [ "services" "matrix" ] host;
  enableTurn = attrsets.hasAttrByPath [ "services" "turn" ] host;
  enableElement = attrsets.hasAttrByPath [ "services" "element" ] host;
  enableAi = attrsets.hasAttrByPath [ "services" "ai" ] host;
};
```

### &#x1F48E; darkone.service.nfs

};

### &#x1F48E; darkone.service.restic

Module de sauvegarde Restic avec configuration DNF.

:::tip
Activer "restic" dans les services d'hôte (config.yaml) lance le serveur REST.
Placez les paramètres de sauvegarde locale dans la configuration nix (conf de la machine par exemple).

```nix
# usr/machines/my-desktop/default.nix
darkone.service.restic = {
  enable = true;
  enableSystemBackup = true;
};
:::

Paramètres par défaut:

```
/srv/nfs/(homes|common) -> /mnt/backup/restic/[host]/srv/nfs
/mnt/medias/(music|videos) -> /mnt/backup/restic/[host]/mnt/medias
/ -> /mnt/backup/restic/[host]/system
```

* **enable** `bool` Activer le service de sauvegarde restic principal
* **enableDryRun** `bool` Mode Dry Run
* **enableServer** `bool` Activer le serveur rest restic
* **enableWaitRemoteFs** `bool` Déclencher le service restic uniquement si le service remote-fs est démarré
* **enableSystemBackup** `bool` Activer la sauvegarde complète du système sauf /srv, /mnt et fichiers cache
* **enableExtraSystemBackup** `bool` Activer la sauvegarde système sur un dépôt supplémentaire
* **mainBackupZone** `str` Nom de la zone de sauvegarde principale (pour sélectionner le bon mot de passe)
* **extraBackupZone** `str` Nom de la zone de sauvegarde supplémentaire (pour sélectionner le bon mot de passe)
* **repositoryRoot** `str` Chemin racine de la cible de sauvegarde principale (local par défaut)
* **extraRepositoryRoot** `str` Chemin racine cible de sauvegarde supplémentaire
* **enableNfsBackup** `bool` Sauvegarder les dirs /srv/nfs/&lt;xxx&gt;
* **enableExtraNfsBackup** `bool` Activer la sauvegarde NFS sur un dépôt supplémentaire
* **nfsPaths** `listOf str` Dirs NFS (/srv/nfs/&lt;xxx&gt;) à inclure dans la configuration de sauvegarde
* **enableMediasBackup** `bool` Sauvegarder les dirs /srv/medias/&lt;xxx&gt;
* **enableExtraMediasBackup** `bool` Activer la sauvegarde médias sur un dépôt supplémentaire
* **mediasPaths** `listOf str` Dirs NFS (/srv/medias/&lt;xxx&gt;) à inclure dans la configuration de sauvegarde

```nix
darkone.service.restic = {
  enable = false;
  enableDryRun = false;
  enableServer = false;
  enableWaitRemoteFs = false;
  enableSystemBackup = false;
  enableExtraSystemBackup = false;
  mainBackupZone = zone.name;
  extraBackupZone = "";
  repositoryRoot = "rest:restic.${zone.domain}:8888";
  extraRepositoryRoot = "rest:restic.${network.zones.other-zone.domain}:8888";
  enableNfsBackup = false;
  enableExtraNfsBackup = false;
  nfsPaths = [ ];
  enableMediasBackup = false;
  enableExtraMediasBackup = false;
  mediasPaths = [ ];
};
```

### &#x1F48E; darkone.system.services

Enregistrement et configuration des services DNF.

:::caution[Module interne spécial]
Ce module est utilisé pour enregistrer et configurer les modules DNF:
- Client du serveur de coordination (headscale / tailscale)
- Proxies inverses (caddy)
- Enregistrements homepage
- Entrées DNS
- Dossiers et fichiers à sauvegarder
:::

* **enable** `bool` Activer le gestionnaire de services DNF pour enregistrer et exposer les services
* **service** `attrs` Configuration globale des services &lt;name&gt;
  * **enable** `bool` Activer le proxy de service
  * **defaultParams** `submodule` Ces options sont calculées par dnfLib.srv.extractServiceParams
    * **domain** `str` Nom de domaine pour le service
    * **title** `str` Nom d'affichage dans homepage
    * **description** `str` Description du service pour homepage
    * **icon** `str` [Nom d'icône pour homepage](https://selfh.st/icons/)
    * **global** `bool` Service global accessible sur Internet
    * **noRobots** `bool` Empêcher les robots de scanner si global est true
    * **fqdn** `str` FQDN calculé ou le service avant le proxy inverse
    * **href** `str` URL calculée du service avant le proxy inverse
    * **ip** `str` IP calculée pour contacter le service
  * **displayOnHomepage** `bool` Afficher un lien sur homepage
  * **persist.dirs** `listOf str` Dossiers persistants du service
  * **persist.files** `listOf str` Fichiers persistants du service
  * **persist.dbDirs** `listOf str` Dossiers persistants du service avec base(s) de données
  * **persist.dbFiles** `listOf str` Fichier(s) de base de données du service
  * **persist.varDirs** `listOf str` Fichiers variables secondaires (log, cache, etc.)
  * **persist.mediaDirs** `listOf str` Dossiers médias du service (images, vidéos, gros fichiers)
  * **proxy.enable** `bool` Créer une configuration virtualHost (false pour les services qui gèrent leur propre configuration)
  * **proxy.isProtected** `bool` Service protégé Oauth2
  * **proxy.isInternal** `bool` Binder le service sur l'interface interne uniquement (pas accessible depuis internet)
  * **proxy.hasReverseProxy** `bool` Ceci est un proxy inverse (ou autre configuration virtualhost via extraConfig)
  * **proxy.defaultService** `bool` Est le service par défaut
  * **proxy.servicePort** `nullOr port` Port interne du service
* **proxy.preExtraConfig** `lines` Configuration Caddy supplémentaire (préfixe pour hôte virtuel)
* **proxy.extraConfig** `lines` Configuration Caddy supplémentaire pour hôte virtuel
  * **proxy.extraGlobalConfig** `lines` Configuration caddy supplémentaire globale
  * **proxy.scheme** `str` Schéma interne du service (http / https)

```nix
darkone.system.services = {
  enable = false;
  service.enable = false;
  service.defaultParams.domain = "";
  service.defaultParams.title = "";
  service.defaultParams.description = "";
  service.defaultParams.icon = "";
  service.defaultParams.global = false;
  service.defaultParams.noRobots = true;
  service.defaultParams.fqdn = "";
  service.defaultParams.href = "";
  service.defaultParams.ip = "";
  service.displayOnHomepage = true;
  service.persist.dirs = [ ];
  service.persist.files = [ ];
  service.persist.dbDirs = [ ];
  service.persist.dbFiles = [ ];
  service.persist.varDirs = [ ];
  service.persist.mediaDirs = [ ];
  service.proxy.enable = true;
  service.proxy.isProtected = false;
  service.proxy.isInternal = false;
  service.proxy.hasReverseProxy = true;
  service.proxy.defaultService = false;
  service.proxy.servicePort = null;
  service.proxy.preExtraConfig = "";
  service.proxy.extraConfig = "";
  service.proxy.extraGlobalConfig = "";
  service.proxy.scheme = "https";
};
```
