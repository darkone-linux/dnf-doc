### &#x1F4BB; darkone.console.git

Un environnement git pré-configuré.

:::tip[Raccourcis Git]
- `git hdf` (history diff) : sélectionner un commit pour afficher le diff.
- `git tree` : arborescence historique.
- `git tre` : historique avec fichiers.
- `git treee` : historique avec messages de commit.
- `git hist` : alternative d'arborescence.
:::

* **enable** `bool` Environnement git pré-configuré

```nix
darkone.console.git.enable = false;
```

### &#x1F4BB; darkone.console.zsh

Une installation zsh complète avec alias, bindkeys et prompt optimisé.

:::tip[Quelques raccourcis]
Général :
- `l`, `la`, `ll` : ls avec options
- `vz` : vim + fzf
- `treef` : arborescence avec fichiers (eza)
- `treed` : arborescence répertoires uniquement (eza)
- `locate` : recherche localsearch (si localsearch est installé au lieu de locate)

Pour le profil admin nix/dnf :
- `nx` : cd /etc/nixos
- `nf` : nixfmt
- `nc` : deadnix
- `dnf` : cd ~/dnf
- `mrproper` : nix-collect-garbage(s) + switch-to-conf boot
:::

* **enable** `bool` Environnement ZSH
* **enableForRoot** `bool` Configuration home manager ZSH pour root

```nix
darkone.console.zsh = {
  enable = false;
  enableForRoot = false;
};
```
