### &#x1F5BC; darkone.graphic.gnome

Environnement GNOME préconfiguré avec dépendances.

* **enable** `bool` Environnement GNOME préconfiguré
* **enableDashToDock** `bool` Extension Dash to dock
* **enableLightDM** `bool` Activer LightDM au lieu de GDM
* **enableCaffeine** `bool` Désactiver la suspension automatique
* **enableGsConnect** `bool` Communication avec les appareils
* **enableOnlineServices** `bool` Comptes en ligne, CalDAV, CardDAV...
* **xkbVariant** `str` Variante de clavier. La disposition est extraite de la console keymap.
* **screenBlankDelay** `ints.unsigned` Délai d'extinction de l'écran en secondes (0 = jamais). Les laptops le fixent à 900 (15 min).

```nix
darkone.graphic.gnome = {
  enable = false;
  enableDashToDock = false;
  enableLightDM = false;
  enableCaffeine = false;
  enableGsConnect = false;
  enableOnlineServices = false;
  xkbVariant = "oss";
  screenBlankDelay = 1800;
};
```

### &#x1F5BC; darkone.graphic.supertuxkart

SuperTuxKart avec les configurations pour jouer sur le réseau local.

:::tip
À utiliser avec le module jeux homemanager !
:::

* **enable** `bool` SuperTuxKart + config pare-feu + partage de pistes
* **isNfsServer** `bool` Serveur NFS (partage de pistes), par défaut le serveur NFS principal. (en cours, activer sur le serveur principal)

```nix
darkone.graphic.supertuxkart = {
  enable = false;
  isNfsServer = isMainNfsServer;
};
```

### &#x1F5BC; darkone.graphic.virt-manager

Gestionnaire Virt + visualiseur avec spice, win-virtio et ovmf.

```nix
darkone.graphic.virt-manager.enable = false;
```

### &#x1F5BC; darkone.graphic.virtualbox

Installation d'hôte Virtualbox.

* **enable** `bool` Installation virtualbox pré-configurée
* **enableExtensionPack** `bool` Activer le pack d'extensions (cause des recompilations)

```nix
darkone.graphic.virtualbox = {
  enable = false;
  enableExtensionPack = false;
};
```
