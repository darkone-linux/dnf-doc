:::

### &#x1F464; darkone.user.build

Générez la configuration NixOS des utilisateurs (pas home-manager).

:::danger[Module obligatoire]
Ce module est activé par défaut (requis par la configuration DNF).
:::

* **enable** `bool` Générateur commun d'utilisateurs (activé par défaut)

```nix
darkone.user.build.enable = true;
```

{/* t:p 62ee13c80b2c575ff0840e6a62a3972455ec323d32fc7402eee9aff1b010040c */}
### &#x1F464; darkone.user.root

Paramètres spécifiques de l'utilisateur root.

:::danger[Module obligatoire]
Ce module est activé par défaut (requis par la configuration DNF).
:::

* **enable** `bool` Configuration spécifique de l'utilisateur root

```nix
darkone.user.root.enable = true;
```
