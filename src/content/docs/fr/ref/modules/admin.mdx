### &#x1F451; darkone.admin.nix

Configuration NixOS pour l'ordinateur administrateur LAN local.

* **enable** `bool` Activer les outils de configuration du générateur NIX
* **enableNh** `bool` Activer l'outil d'aide nix (nh)

```nix
darkone.admin.nix = {
  enable = false;
  enableNh = false;
};
```
