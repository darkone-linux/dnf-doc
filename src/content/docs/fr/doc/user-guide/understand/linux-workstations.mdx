Le système Linux du cloud s'installe sur n'importe quel ordinateur PC relié au réseau.

:::note[Linux, c'est compliqué ?]
Contrairement à ce qu'on croit, **Linux et ses applications sont devenus très simples d'utilisation.**
Et notre système pré-installé est encore plus facile à prendre en main&nbsp;!

**Pourquoi Linux / NixOS ?**

- **Linux** est libre, sûr, en raccord avec l'idée d'un espace numérique souverain et sécurisé.
- **NixOS** est un Linux innovant : déclaratif, reproductible et très stable.
- Un code ouvert et maintenu garantit le risque de "backdoors" le plus faible au monde.
:::

:::tip[Un gain de temps à tous les niveaux]
- 100% pré-installé → programmes, comptes utilisateur et configurations, tout y est.
- 100% info-gérés → pas besoin d'administrer le système, tout est automatique.
- Idéal et performant avec les vieux PC qu'on utilise plus.
:::

## Les profils de postes

Chaque machine est configurée selon un **profil** (rôle) comportant :

- Des **fonctionnalités** activées automatiquement.
- Des **configurations** toutes faites.

| | Profil | Rôle |
|---|---|---|
| ⚙️ | `minimal` | Un système avec le strict minimum. |
| 🖥️ | `desktop` | Un poste fixe avec des logiciels bureautique, multimédia, productivité. |
| 💻 | `laptop` | Un poste bureautique optimisé pour ordinateur portable. |
| 🔌 | `portable` | Mon système sur une clé USB amorçable. |
| 🗄️ | `server` | Un serveur élagué et sécurisé, sans écran. |
| 🌐 | `gateway` | Une passerelle de zone, ma "box" internet qui filtre les pubs. |
| ☁️ | `hcs` | Un serveur de coordination du VPN, pour l'administrateur. |
| 📦 | `vm` | Une machine virtuelle. |

Détails techniques&nbsp;: [Profils d'hôtes](/fr/doc/admin-guide/understand/host-profiles/).

:::tip[Mon profil est adaptable !]
- Je peux le choisir à l'installation et plus tard, **le changer à tout moment**.
- Je peux y ajouter des **programmes et configurations personnelles**.
:::

## Zéro maintenance

Sous DNF, le système s'administre "tout seul"&nbsp;:

- **Mises à jour** : appliquées automatiquement, sans intervention.
- **Sécurité** : Linux est sûr par conception → pas besoin d'antivirus.
- **Pilotes** : tout est pré-configuré. Imprimantes, écrans, périphériques&nbsp;: ça marche.
- **Panne ou problème** : l'administrateur restaure votre poste en quelques minutes, à l'identique.

:::tip[Je veux réinstaller]
Changement d'ordinateur, vol ou problème technique grave ? On peut tout réinstaller&nbsp;:

- Mes logiciels
- Mes configurations
- Mes données

⚠️ A condition que ma configuration soit déclarée et que mes données soient sauvegardées.
:::
