## M'identifier

J’ai besoin :

* **D’un mot de passe principal** pour mon coffre-fort (vaultwarden).
* **D’une identité unique** pour tous les services du cloud (kanidm / IDM).

Mon administrateur doit effectuer une procédure pour que je puisse définir ces accès.

* **Mot de passe principal** : un e-mail contenant un lien pour saisir le mot de passe.
* **Identité** : un QR-Code ou un lien pour définir une clé et un mot de passe "2FA".

:::tip[À savoir !]
- 🔒 **Mon mot de passe principal** (Vaultwarden / Bitwarden)<br/>permet de me connecter à mon coffre-fort.
- 🔑 **Une clé d’authentification** (Clé IDM ou [YubiKey](#utiliser-ma-yubikey))<br/>permet de me connecter à toutes les applications du cloud.
- 🔐 **Un 2ème facteur d'authentification** ([2FA](#ajouter-un-2ème-facteur))<br/>permet de renforcer un accès par mot de passe avec une étape supplémentaire.
:::

Le **mot de passe principal** est primordial, c'est le seul qu'il faut **absolument retenir**.

## Utiliser ma YubiKey

Ma YubiKey est une clé physique qui remplace le mot de passe dans la plupart
des situations :

- **Ordinateur chiffré** : au démarrage, je touche la YubiKey quand la
  diode clignote → le disque se déverrouille tout seul, sans passphrase.
- **Connexion à ma session** : je touche la YubiKey sur l'écran de
  connexion au lieu de taper mon mot de passe.
- **Kanidm (SSO)** : mon administrateur me transmet un lien
  d'enregistrement. Je le visite une fois, je crée ma passkey avec la YubiKey
  → ensuite, je touche la clé pour me connecter à tous les services du cloud.

:::note[Qu'est-ce qu'une clé ?]
Une clé numérique permet de s’identifier sans avoir à utiliser de mot de passe.
Elle est située dans mon coffre-fort.
Quand une application en a besoin, Bitwarden s’ouvre automatiquement.
C’est le moyen **LE PLUS SIMPLE ET SÉCURISÉ** pour m’authentifier.
:::

:::tip[YubiKey = le plus simple]
Une fois enregistrée, je n'ai plus qu'à toucher ma clé. Pas de mot de passe à
retenir ni de code à recopier.
:::

## Ajouter un 2ème facteur

On ajoute souvent un 2ème facteur à un mot de passe, pour le sécuriser.
De nos jours, énormément de mots de passe sont compromis, c’est pourquoi ce type
d’authentification devient nécessaire.

:::tip[Une authentification 2 facteurs (2FA) associe]
- Un mot de passe (que je connais)
- ET un autre “facteur” (que je possède)
:::

Le 2ème facteur peut être un code envoyé par mail / sms, un générateur de code à 6 chiffres, etc.

:::caution[Sécuriser mon coffre-fort Vaultwarden]
Un mot de passe principal “simple” est autorisé par défaut.
Mais il est vivement recommandé de lui adjoindre un 2ème facteur pour sécuriser l’accès,
surtout si je mets tous mes mots de passe dedans (banque, comptes sociaux, etc.).
:::
